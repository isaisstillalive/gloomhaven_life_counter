- 最左列「＋」ボタンのシフトクリックでキャラクター追加。
- 最左列「＋」ボタンのクリックで全体をダメージ表記に変更。
- 最上段イラストのダブルクリックでキャラクター＆データ変更。

- 「N」で通常モンスター追加、「E」で上級モンスター追加。
- ライフカウンターは左クリックで減少、右クリックで増加。
- ライフカウンターの[-]ボタンクリックでモンスター削除。
- 最左列「速」ボタンクリックで行動順位リセット。
- 「速」段の数字クリックで順番に行動順位を設定。
